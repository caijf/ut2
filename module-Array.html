(static) nth(array, nopt) → {*}

Description:

  • 获取数组的第 n 个元素。如果 n 为负数，则返回从数组结尾开始的第 n 个元素。

    同 Array.prototype.at 方法。

Source:
  • nth.js, line 23
Since:
  • 1.0.0
Example
const arr = ['a', 'b', 'c', 'd'];

nth(arr, 1); // 'b'

nth(arr, -2); // 'c'
