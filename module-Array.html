(static) chunk(array, sizeopt) → {Array}

Description:

  • 将数组拆分成多个 size 长度的区块，并将这些区块组成一个新数组。

    如果数组无法被分割成全部等长的区块，那么最后剩余的元素将组成一个区块。

Source:
  • chunk.js, line 22
Since:
  • 1.0.0
Example
const array = ['a', 'b', 'c', 'd'];

chunk(array, 2); // [['a', 'b'], ['c', 'd']]

chunk(array, 3); // [['a', 'b', 'c'], ['d']]

(static) difference(array, values, iteratee)

Description:

  • 创建一个 array 排除 values 值的新数组。使用了 SameValueZero 做等值比较。如果传入迭代函数，会调用数组的每个元素以产生唯一性计算的标准。

    iteratee 调用时会传入 1 个参数 value 。

Source:
  • difference.js, line 28
Since:
  • 1.0.0
Example
difference([3, 1, 2], [4, 2]); // [3, 1]

difference([3.1, 2.2, 1.3], [4.4, 2.5], Math.floor); // [3.1, 1.3]

difference([{x: 2}, {x: 1}], [{x: 1}], item=>item.x); // [{x: 2}]

// 如果迭代元素为对象，迭代函数可以直接写入对象属性。
difference([{x: 2}, {x: 1}], [{x: 1}], 'x'); // [{x: 2}]
Parameters:
Name Type Description
array

要检查的数组。
values

排除的值。
iteratee

迭代函数，调用每个元素。
Returns:

过滤值后的新数组。

(static) uniq(array, iterateeopt) → {Array}

Description:

  • 创建一个去重后的数组副本。使用了 SameValueZero 做等值比较。只有第一次出现的元素才会被保留。如果传入迭代函数，会调用数组的每个元素以产生唯一性计算的标准。

    iteratee 调用时会传入 1 个参数 value 。

Source:
  • uniq.js, line 27
Since:
  • 1.0.0
Example
uniq([2, 1, 2]); // [2, 1]

uniq(['a', NaN, 2, 1, NaN, 'a', 1]); // ['a', NaN, 2, 1]

uniq([{x: 1}, {x: 2}, {x: 1}], item=>item.x); // [{x: 1}, {x: 2}]

// 如果迭代元素为对象，迭代函数可以直接写入对象属性。
uniq([{x: 1}, {x: 2}, {x: 1}], 'x'); // [{x: 1}, {x: 2}]
Parameters:
