(static) inRange(number, start, endopt) → {number}

Description:

  • 检查数字是否在 start 与 end 之间，但不包括 end 。

    如果 start 大于 end 会自动交换范围参数。

Source:
  • inRange.js, line 35
Since:
  • 1.0.0
Example
inRange(3, 2, 4); // true

inRange(1, 0, 2); // true

inRange(1.2, 0.5, 1.5); // true

inRange(2.2, 0.5, 1.5); // false

inRange(-2, -2, 4); // true

inRange(4, -2, 4); // false

// 某个范围为 0 可省略最后一个参数
inRange(1， 2); // true

inRange(1， -2); // false
