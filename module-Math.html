(static) max(array, iterateeopt) → {*}

Description:

  • 调用 array 中的每一个元素，来生成其值排序的标准，返回最大的值。

Source:
  • max.js, line 27
Since:
  • 1.0.0
Example
const array = [1, 2, 3];

max(array); // 3

const objects = [{n: 1}, {n: 2}];

max(objects, item => item.n); // {n: 2}

// 迭代函数可以直接写入属性。
max(objects, 'n'); // {n: 2};
